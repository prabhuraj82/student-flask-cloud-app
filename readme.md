# Cloud Ready Flask Application

## Overview
This is a lightweight Flask web application designed to demonstrate cloud deployment readiness.

## Features
- Python Flask backend
- Simple web interface
- Ready to deploy on AWS EC2
- Can integrate AWS S3 for static file/document storage
- Scalable architecture using Gunicorn + Linux server

## AWS Deployment Approach
1. Launch EC2 instance
2. Install Python dependencies
3. Run Flask app using Gunicorn
4. Configure security groups
5. Optional: Use AWS S3 for file uploads/static assets

## Technologies Used
- Python
- Flask
- HTML
- AWS EC2 (deployment-ready)
- AWS S3 (storage concept)
